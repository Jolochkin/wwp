有些 bug

  • 发布出来的内容不应该有 ASCII
  • 单行输入框不应该换行
  • 读者打不开文章

需要补充

  • 先学习一个单词，下一条用到：caret = 光标
  • 单行输入框按「上」，caret 移到前一个输入框的尾部。
  • 单行输入框按「下」或「回车」，caret 移到后一个输入框的头部。
  • 多行输入框的第一行按「上」，caret 移到前一个输入框的尾部。
  • 多行输入框的最后一行按「下」，caret 移到后一个输入框的头部。
  • 单行输入框目前包括：作者栏，标题栏
  • 多行输入框目前包括：正文栏
  • 正文中，每次回车创建一个 block 对象，block 类型默认为 p。caret 位置：<div><p>|<p/><div/>
  • 作者本人看到文章可以编辑。非作者看不到编辑按钮。
